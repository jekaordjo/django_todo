# Django Todo App 📝

Простой и функциональный менеджер задач на Django. Проект создан для практики построения архитектуры веб-приложений и управления зависимостями через `uv`.

## 🛠 Технологии
- **Python 3.13+**
- **Django 6.x**
- **uv** (управление пакетами и окружением)
- **SQLite** (база данных по умолчанию)
